Conversion functions

Stuff for converting between various useful things
atpToUniversal (square, easting, southing)
Converts standard three-parameter ATP coordinates to screen coordinates

Parameters:

  • square
    • The ATP grid square we need to correct for.
  • easting
    • Easting within the given ATP square.
  • southing
    • Southing within the given ATP square.

Returns:

    Screen coordinates as int in format easting, southing.
cardinalToHeading (cardinalDirection)
Converts a cardinal-direction string to heading in degrees.

Parameters:

  • cardinalDirection
    • Direction as string, e.g. "north", "e", "south-southwest", "ene"

Returns:

    Heading in degrees as float
headingToCardinal8 (inputHeading)
Finds approximate 8-point compass direction from input heading.

Parameters:

  • inputHeading
    • Heading to convert, int or float.

Returns:

    Nearest 8-point compass direction as string.
headingToCardinal16 (inputHeading)
Finds approximate 16-point compass direction from input heading.

Navigation functions

The core functions that allow for navigational calculations.
adjustCoords (dir, inputEasting, inputSouthing)
Returns current coordinates adjusted for input direction.

Parameters:

  • dir
    • string Short-form direction string.
  • inputEasting
    • int Current Navigator easting.
  • inputSouthing
    • int Current Navigator southing.

Returns:

    int Easting, southing as screen coordinates.

Usage:

    mymyNavigator:adjustCoords("n",625,715)
findWaypoint (targetWaypoint)
Finds bearing and distance to waypoint from current location.

Parameters:

  • targetWaypoint
    • string Shortname of the waypoint to located.

Returns:

    • int Bearing to waypoint
      * int Chebyshev distance
getBearing (originX, originY, targetX, targetY)
Returns the bearing to target coordinates from origin coordinates.

Parameters:

  • originX
  • originY
  • targetX
  • targetY Predeclarations

Returns:

    Bearing to target coordinate in degrees, float.
getChebyshevDistance (originX, originY, targetX, targetY)
Returns the Chebyshev distance between two points. Does not account for impassable seas or obstacles.

Parameters:

  • originX
    • int Easting of the origin point.
  • originY
    • int Southing of the origin point.
  • targetX
    • int Easting of the destination point.
  • targetY
    • int Southing of the destination point.

Returns:

    int Chebyshev distance as an integer.
getNearestHarbours (numReturns)
Locates an arbitrary number of the nearest harbours. Gathers the name, distance, and bearings of the nearest harbours, sorted by distance. Limited in number of returns by numReturns.

Parameters:

  • numReturns
    • int The number of nearest harbours to return results for.

Returns:

    table Indexed table containing information for the numReturns nearest harbours, sorted by ascending distance.
