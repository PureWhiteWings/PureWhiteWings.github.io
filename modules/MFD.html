Module MFD

Display for general shipboard use.

Mimics a 16x52 DOS-esque terminal, because I like the aesthetic.

Initialisation methods

For creating and Initialising MFD object.
build (mfd_name, mfd_X, mfd_Y)
Creates and blanks the MFD display.

Parameters:

  • mfd_name
    • string Name for the consoles and such. Should be identical to object's unique name.
  • mfd_X
    • int X screen coordinate of MFD's top-left point.
  • mfd_Y
    • int Y screen coordinate of MFD's top-left point.

Returns:

    bool True if successful.

Usage:

    myMFD:build("myMFD",240,632)

Notification methods

For drawing notifications on-screen.
notify (text, width, height, displayTime, fontSize, bgColour, fgColour)
Raises an on-screen notification, with multiple options for styling. Raises a notification on screen, centered in lower-middle.

Parameters:

  • text
    • string The notification text to display.
  • width
    • int Width of the notification.
  • height
    • int Height of the notification
  • displayTime
    • int How many seconds to show the notification. 0 leaves notification indefinitely.
  • fontSize
    • int The font size to use (default 12).
  • bgColour
    • [optional] string Colour to use for alert background, as hex string.
  • fgColour
    • [optional] string Colour to use for foreground text, as HTML colour name.

Returns:

    None.

Usage:

    myMFD:notify("Testing...", 160, 50, 2, 12, "#004400", "yellow")

Query methods

Stuff for retrieving data to help us draw to the screen.
getColour ()
Retrieves current time of day from Time object, if present. Defaults to green if no Time object.

Returns:

    HTML colour string, depending on Time:isNight().

Usage:

    html_colour = MFD:getColour()

Output methods

Stuff for outputting to the MFD screen.
cls ()
Resets and erases the entire screen, and readies for writing. Clears the entirety of the consoles, and fills it with spaces so write functions can successfully locate.

Returns:

    None.

Usage:

    myMFD:cls() myMFD:drawBorder()
drawBorder (colour)
Draws a basic border around the edges of the screen. Will overwrite any text beneath the border!

Parameters:

  • colour
    • [optional] string Colour to draw border in, as HTML colour name.

Returns:

    None.

Usage:

    myMFD:drawBorder("cyan")
drawBorderNav (colour)
Draws a basic border around the edges of the screen, arranged for Nav page. Will overwrite any text beneath the border!

Parameters:

  • colour
    • [optional] string Colour to draw border in, as HTML colour name.

Returns:

    None.

Usage:

    myMFD:drawBorderNav("forest_green")
setCursor (inputX, inputY, doBlink)
Places an underscore at the given X,Y with optional blinking. Purely aesthetic, serves no functional purpose.

Parameters:

  • inputX
    • num X coordinate of cursor position.
  • inputY
    • num Y coordinate of cursor position.
  • doBlink
    • bool Blink cursor at the rate of cursorRate if true.

Returns:

    None.

Usage:

    myMFD:setCursor(25,12,true)
write (writeText, writeX, writeY, fgColour, bgColour)
Writes text to the MFD. Writes text of arbitrary length, position, and colour to the MFD. Overwrites existing text for the length of the string.

Parameters:

  • writeText
    • string The text to be written to the MFD.
  • writeX
    • int X position of cursor, where text will be inserted. 0-51
  • writeY
    • int Y position of cursor, where text will be inserted. 0-15
  • fgColour
    • [optional] string Colour to be used for the text foreground colour, as HTML colour name. Defaults to standard day/night colour if not specified.
  • bgColour
    • [optional] string Colour to be used for the text background colour, as HTML colour name. Defaults to *near_black* if not specified.

Returns:

    bool True if write successful.

Usage:

    myMFD:write("Running calibration...", 2, 2, "green", "dim_grey")
