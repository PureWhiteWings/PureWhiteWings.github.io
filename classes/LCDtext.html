Class LCDtext

Gauges that sit at the bottom and do gauge things related to things to do on a ship.

Initialisation

LCDtext:new (_name, _elementX, _elementY, _inputText, _backingText, int_fontSize[, Font])
Creates a new instance of the LCDtext class.

Parameters:

  • _name string Instance name, should be identical to variable instance is assigned to.
  • _elementX int
  • _elementY int
  • _inputText string
  • _backingText test Dark backing text to mimic unlit LCD segments.
  • int_fontSize {opt] Default 30
  • Font string_fontColour colour in a QtCSS compatible string. Default HotPink. (optional)

Returns:

    table A new instance of the LCDtext class.

Usage:

    lcd = LCDtext:new("lcd")

Misc methods

LCDtext:setVisible (_visible)
Shows or hides the whole object on the screen. Just a shortcut for things like getting into my ship UI.

Parameters:

  • _visible bool True to show object, false to hide.

Usage:

    LCDtext:setVisible(true)
